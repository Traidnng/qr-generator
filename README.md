# 🚀 CLI QR Code Generator

A lightweight, bilingually supported (EN/TR) Command Line Interface (CLI) QR Code generator written in Python. It features custom color configurations, history logging, and real-time ASCII previews inside the terminal.

## ✨ Features
* **Bilingual Support:** Easily switch between English and Turkish.
* **ASCII Preview:** See the QR code directly in your terminal before saving.
* **Custom Styles:** Choose from pre-defined color schemes or get a random style.
* **History Management:** Automatically keeps track of the last 50 generated QR codes in a local JSON file.
* **Auto-Open:** Instantly view the latest generated image with platform-specific system triggers.
